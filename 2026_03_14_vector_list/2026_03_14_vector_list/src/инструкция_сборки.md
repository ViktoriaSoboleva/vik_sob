# Инструкция по сборке и запуску тестов на Windows

## Требования

- Установленный **MSYS2** — https://www.msys2.org/
- Установленный **MinGW** (g++) — уже должен быть, если компилировали `main.cpp`

---

## Шаг 1 — Установить зависимости в MSYS2

Открыть **MSYS2 MinGW x64** из меню Пуск.  
В строке приглашения должно быть написано `MINGW64`.

Выполнить:

```bash
pacman -S mingw-w64-x86_64-cmake mingw-w64-x86_64-gcc mingw-w64-x86_64-make mingw-w64-x86_64-gtest
```

На вопрос `[Y/n]` нажать `Y` и Enter.

---

## Шаг 2 — Перейти в папку проекта

В том же окне MSYS2 MinGW64:

```bash
cd /c/Users/vsv10/Downloads/2026_03_14_vector_list_done/2026_03_14_vector_list
```

> Путь к папке пишется через `/` вместо `\`, и диск — `/c/` вместо `C:\`

---

## Шаг 3 — Создать папку сборки и сгенерировать файлы

```bash
mkdir build
cd build
/mingw64/bin/cmake .. -G "MinGW Makefiles" -Wno-dev \
  -DCMAKE_MAKE_PROGRAM=/mingw64/bin/mingw32-make.exe \
  -DCMAKE_C_COMPILER=/mingw64/bin/gcc.exe \
  -DCMAKE_CXX_COMPILER=/mingw64/bin/g++.exe
```

Ожидаемый результат:
```
-- Configuring done
-- Generating done
-- Build files have been written to: ...
```

---

## Шаг 4 — Собрать проект

```bash
/mingw64/bin/mingw32-make.exe
```

Ожидаемый результат:
```
[100%] Built target containers_lib
[100%] Built target doubly_linked_list_tests
[100%] Built target vector_tests
```

---

## Шаг 5 — Запустить тесты

```bash
cd test
./vector_tests.exe
./doubly_linked_list_tests.exe
```

Ожидаемый результат:
```
[  PASSED  ] 8 tests.   ← Vector
[  PASSED  ] 8 tests.   ← DoublyLinkedList
```

---

## Примечания

- Все команды выполнять **только в окне MSYS2 MinGW64**, не в обычном терминале Windows
- Если папка `build` уже существует и что-то пошло не так, очистить её: `rm -rf *` (находясь внутри `build`)
- Повторная сборка после изменений в коде: просто снова запустить `/mingw64/bin/mingw32-make.exe` из папки `build`
